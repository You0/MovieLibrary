# MovieLibrary


# 简介
一款比较简单的电影检索APP，就是可以根据关键字，演员，类型或者评分去筛选出电影,直接查看预告片和剧照，简介，演员等等的信息。
这款app是比较简单，但是感觉这是我独立完成程度最高的一款app（从数据爬取到后端搭建，到app开发一条龙），所以想介绍一下。
其中衍生了很多工具，很具有锻炼意义。
1、数据源获取，就是利用爬虫框架把豆瓣电影爬下来 爬虫框架链接：
2、然后自己搭建web后台，优化数据库  后端代码链接：
